# Python Port Scanner

A fast multithreaded port scanner built using Python.

## Features
- Scans custom IP or domain
- Custom port range input
- Multithreaded (fast scanning)
- Displays open ports only

## Technologies Used
- Python
- socket
- threading

## How to Run

1. Clone the repository
2. Run the script:

```bash
python port_scanner.py
```
3. Enter:
-Target IP/domain
-Start port
-End port

## Example output
scanme.nmap.org:22 is open
